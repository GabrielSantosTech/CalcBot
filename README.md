# 🤖 CalcBot Ultimate — IA Matemática

O **CalcBot Ultimate** é um chatbot inteligente especializado em cálculos matemáticos. Com uma interface futurista e focada em experiência do usuário (UX), ele permite realizar operações complexas através de linguagem natural ou via um teclado numérico otimizado para dispositivos móveis.

![Preview do CalcBot](https://mteasy.netlify.app/)

## ✨ Principais Diferenciais

* **🗣️ Interface Conversacional:** Digite como se estivesse conversando. O bot entende "mais", "vezes", "dividido por", "raiz de", entre outros.
* **📱 Mobile Optimized:** Inclui um teclado numérico customizado que resolve o problema de teclados nativos de sistemas mobile, além de gestos como "swipe" para abrir o menu.
* **📊 Painel de Estatísticas:** Acompanhe em tempo real o número de mensagens, cálculos feitos e a operação que você mais utiliza na sessão.
* **📜 Histórico Inteligente:** Clique em qualquer cálculo anterior para carregá-lo novamente no campo de entrada.
* **🎨 Design Ultra-Moderno:** Estética baseada em camadas de vidro, gradientes vibrantes e suporte total a temas (Dark/Light).
* **📋 Atalhos Rápidos:** Copie resultados instantaneamente com um clique duplo ou pressão longa no celular.

## 🛠️ Tecnologias de Elite

Este projeto foi construído "pure code" para garantir máxima performance e zero dependência de bibliotecas pesadas:

* **HTML5 Semântico:** Estrutura otimizada para acessibilidade e SEO.
* **CSS3 Avançado:** Uso intenso de `variables`, `flexbox`, `grid`, `animations` e `backdrop-filter` para o efeito glass.
* **Vanilla JavaScript (ES6+):** Lógica de processamento de strings (Regex), manipulação de DOM em tempo real e APIs de vibração e área de transferência.

## ⚙️ Funcionalidades Matemáticas Suportadas

* Aritmética básica (Adição, Subtração, Multiplicação, Divisão).
* Cálculos de Porcentagem e Resto.
* Potenciação e Raiz Quadrada/Cúbica.
* Fatorial (até 20!).
* Logaritmos e Log Natural (ln).
* Conversão para decimal.
